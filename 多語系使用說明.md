# 多語系系統使用說明

## 🌍 功能說明

您的網站現在支援**中文（繁體）**和**英文**雙語切換！

### 特點
- ✅ 自動檢測瀏覽器語言
- ✅ 記住用戶選擇的語言
- ✅ 點擊即時切換，無需重新載入頁面
- ✅ 支援 URL 參數（例如：`?lang=en`）
- ✅ 響應式設計，手機也能輕鬆切換

---

## 🎯 如何使用

### 1. 用戶切換語言

**桌面版：**
- 在頂部導航欄右側，會看到「中」和「EN」兩個按鈕
- 點擊即可切換語言

**手機版：**
- 語言切換按鈕在漢堡選單按鈕旁邊
- 點擊切換

### 2. 語言記憶功能

- 用戶選擇的語言會儲存在瀏覽器
- 下次訪問會自動使用上次的語言

### 3. 分享特定語言的連結

在網址後面加上語言參數：
```
https://your-website.com/?lang=en  (英文)
https://your-website.com/?lang=zh-TW  (中文)
```

## 📝 如何為頁面添加翻譯

### 步驟 1：在 HTML 中添加標記

為需要翻譯的元素添加 `data-i18n` 屬性：

```html
<!-- 原本的 -->
<h1>專業、可靠的租賃服務</h1>

<!-- 改為 -->
<h1 data-i18n="hero.title">專業、可靠的租賃服務</h1>
```

### 步驟 2：在 i18n.js 中添加翻譯

編輯 `js/i18n.js` 文件，在 `translations` 物件中添加：

```javascript
const translations = {
  'zh-TW': {
    'hero.title': '專業、可靠的租賃服務',
    'hero.subtitle': '機場接送、旅遊包車...',
    // 更多翻譯...
  },
  'en': {
    'hero.title': 'Professional & Reliable Rental Services',
    'hero.subtitle': 'Airport transfers, tour packages...',
    // 更多翻譯...
  }
};
```

### 步驟 3：引入 i18n.js

在頁面底部引入腳本：

```html
<script src="js/i18n.js"></script>
```

## 🔧 為其他頁面添加多語系

### 1. 貨車出租頁面（貨車出租.html）

**添加語言切換按鈕：**
```html
<div class="language-switcher">
  <button class="lang-btn active" data-lang="zh-TW" title="中文">中</button>
  <button class="lang-btn" data-lang="en" title="English">EN</button>
</div>
```

**添加翻譯標記：**
```html
<h1 data-i18n="truck.title">貨車出租</h1>
<p data-i18n="truck.description">自助搬家、貨物運輸的最佳幫手。</p>
```

**在 i18n.js 添加翻譯：**
```javascript
'zh-TW': {
  'truck.title': '貨車出租',
  'truck.description': '自助搬家、貨物運輸的最佳幫手。',
},
'en': {
  'truck.title': 'Truck Rental',
  'truck.description': 'Perfect for moving and cargo transport.',
}
```

### 2. 露營車出租頁面（露營車出租.html）

同樣的方式：
1. 添加語言切換按鈕
2. 為文字添加 `data-i18n` 屬性
3. 在 i18n.js 添加翻譯
4. 引入 i18n.js 腳本

## 📋 翻譯命名規則

建議使用以下命名格式：

```
區塊.元素.類型
```

### 範例：

| 翻譯 key | 說明 |
|---------|------|
| `nav.home` | 導航欄的「首頁」 |
| `hero.title` | Hero 區塊的標題 |
| `service.airport.desc` | 機場接送服務的描述 |
| `footer.copyright` | Footer 的版權文字 |
| `btn.book-now` | 「立即預約」按鈕 |

## 💡 進階功能

### 1. 監聽語言切換事件

```javascript
window.addEventListener('languageChanged', (e) => {
  console.log('語言已切換為：', e.detail.lang);
  // 執行其他操作...
});
```

### 2. 以程式方式切換語言

```javascript
// 切換到英文
i18n.switchLanguage('en');

// 切換到中文
i18n.switchLanguage('zh-TW');
```

### 3. 獲取當前語言

```javascript
const currentLang = i18n.currentLang;
console.log('當前語言：', currentLang);
```

## 🐛 常見問題

### Q1: 為什麼切換語言後，某些文字沒有改變？

**A:** 確認該元素是否有添加 `data-i18n` 屬性，以及翻譯 key 是否正確。

### Q2: 如何添加更多語言（例如日文）？

**A:** 在 `i18n.js` 的 `translations` 物件中添加新語言：

```javascript
const translations = {
  'zh-TW': { /* 中文翻譯 */ },
  'en': { /* 英文翻譯 */ },
  'ja': { /* 日文翻譯 */ }  // 新增
};
```

然後添加對應的語言按鈕：
```html
<button class="lang-btn" data-lang="ja" title="日本語">日</button>
```

### Q3: 語言切換後，頁面會重新載入嗎？

**A:** 不會！語言切換是即時的，使用 JavaScript 動態更新文字內容，無需重新載入。
